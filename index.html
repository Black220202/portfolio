04. Projects

Featured Work

Selected projects showcasing technical depth and design execution

Restaurant & Hospitality

L'ÉPOQUE — Fine Dining Experience

A sophisticated fine dining restaurant website with an elegant dark aesthetic, immersive imagery, and seamless reservation flow.

Challenge

Creating a luxury digital presence that mirrors the exclusivity of a Michelin-starred dining experience while maintaining blazing-fast performance across all devices.

Solution

Built a custom animation system using IntersectionObserver and CSS transforms for buttery-smooth scroll-triggered reveals. Implemented lazy-loaded, WebP-optimized imagery with LQIP placeholders. Designed an accessible, keyboard-navigable reservation flow with real-time availability checking.

HTML5 CSS3 Vanilla JS IntersectionObserver WebP Optimization
Architecture & Interior Design

AURA — Interior Design Studio

A minimalist luxury interior design & architecture studio website with clean aesthetics and a refined neutral palette.

Challenge

Translating high-end print-design sensibilities to the web — maintaining typographic precision, whitespace harmony, and image quality across breakpoints without sacrificing performance.

Solution

Developed a fluid typography system using CSS clamp() and custom properties for seamless scaling. Built a Masonry-style project grid with CSS Grid and auto-placement. Implemented a custom image comparison slider for before/after project showcases using pure CSS.

HTML5 CSS3 Vanilla JS CSS Grid Fluid Typography
Product & Audio

Aura Pro — Sound, Reimagined

A sleek, dark-themed product landing page for premium audio equipment with futuristic design and bold typography.

Challenge

Building a high-impact product page that communicates premium audio quality through visual and interactive design — including a performant 3D product viewer and scroll-synchronized narrative sections.

Solution

Integrated a lightweight Three.js product viewer with DRACO-compressed GLTF models (<50KB). Created a scroll-driven animation timeline using GSAP ScrollTrigger for cinematic section transitions. Built a custom audio visualizer using Web Audio API and Canvas for the hero section.

HTML5 CSS3 Three.js GSAP Web Audio API
